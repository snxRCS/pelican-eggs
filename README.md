# pelican-eggs

Community-maintained [Pelican Panel](https://pelican.dev) eggs by [@snxRCS](https://github.com/snxRCS).

## Available eggs

### Voice

| Egg | Description | Source |
|---|---|---|
| [TS6-Manager](voice/TS6-Manager-EGG) | All-in-one egg for [clusterzx/ts6-manager](https://github.com/clusterzx/ts6-manager) — backend + React frontend + Go WebRTC sidecar supervised in one container. | [egg JSON](voice/TS6-Manager-EGG/egg-teamspeak6-manager.json) |

## How to use

In your Pelican panel admin area:

**Eggs → Import → URL** and paste the raw URL of the egg JSON, e.g.:

```
https://raw.githubusercontent.com/snxRCS/pelican-eggs/main/voice/TS6-Manager-EGG/egg-teamspeak6-manager.json
```
